Module pal::vmcs::guest_interruptibility_state[−][src]

Expand description

Guest interruptibility state.

The IA-32 architecture includes features that permit certain events to be blocked for a period of time. This field contains information about such blocking. Details and the format of this field are given in Table 24-3.
